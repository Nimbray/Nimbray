# Agent Inbox

Ce dossier sert aux nouvelles conversations ou aux nouveaux agents qui doivent travailler sans écraser les autres.

Créer un espace :

```bash
npm run source:agent -- --agent nom-agent
```

Chaque dossier créé contient un `HANDOFF.md` avec la source de départ, les fichiers modifiés et les tests à compléter avant intégration officielle.
