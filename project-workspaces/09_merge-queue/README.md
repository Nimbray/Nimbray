# Merge Queue

Ce dossier sert à préparer les fusions entre agents.

Chaque PR importante peut ajouter une fiche ici avec :

- agent concerné ;
- branche ;
- fichiers touchés ;
- dépendances avec d’autres agents ;
- risques ;
- décision intégrateur.

Seul l’Agent Intégrateur transforme une série de contributions en nouvelle version officielle.
