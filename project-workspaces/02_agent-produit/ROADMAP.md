# Roadmap Produit

## Priorité immédiate

- Stabiliser les usages fondamentaux : chat, upload, mémoire, réponses naturelles.
- Garder le produit simple, rapide et compréhensible.
- Transformer chaque idée en priorité : maintenant, ensuite, plus tard.

## Critères d'acceptation

- L'utilisateur comprend ce qui a changé.
- Le changement améliore vraiment l'expérience.
- Le projet reste déployable sur Vercel.
