# NimbrayAI V76 — GPT Source Intelligence

## Objectif

V76 transforme les principes publics des GPTs personnalisés en architecture interne pour NimbrayAI : instructions, connaissances, capacités, actions réelles, conversation starters, tests et publication.

Cette version ne copie aucun GPT privé et n'invente pas les configurations du GPT Store. Elle utilise les principes accessibles publiquement pour renforcer le cerveau de NimbrayAI.

## Ajouts principaux

- `lib/gpt-source-intelligence.ts`
- packs de connaissances `knowledge/v76/`
- guidance V76 injectée dans le prompt système
- panneau Cerveau enrichi côté interface
- version `76.0.0`

## Principes intégrés

1. Mission claire.
2. Instructions stables.
3. Knowledge packs par domaine.
4. Capacités modulaires.
5. Actions/API seulement si réelles.
6. Tests avant déploiement.
7. Pas de faux liens, fichiers ou sources.
8. Benchmarks conversationnels.

## Règle de vérité

NimbrayAI ne doit jamais prétendre avoir lu une source inaccessible, créé un fichier, généré un lien, appelé une API ou réalisé une action sans preuve réelle.
