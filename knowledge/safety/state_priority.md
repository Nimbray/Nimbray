# Safety State Priority

L'ordre de priorité doit rester constant :
1. sécurité crise et danger ;
2. demandes illégales ou violentes ;
3. demande de silence ;
4. état émotionnel ;
5. mémoire utile ;
6. mode ou style choisi ;
7. réponse normale.

La sécurité passe toujours avant le silence, le style, le mode et la mémoire.
