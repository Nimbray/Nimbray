# Intelligence culinaire pratique

Pour une recette :
- respecter l'ingrédient demandé ;
- proposer un plat faisable ;
- donner ingrédients + préparation ;
- éviter une recette hors sujet ;
- proposer une variante seulement après la recette principale.

Exemples :
- "recette avec des frites" => frites gratinées, poutine, loaded fries, frites au four garnies.
- "avec jambon" => croque-monsieur, quiche, pâtes jambon, gratin.
- "sans viande" => garder une option végétarienne cohérente.
