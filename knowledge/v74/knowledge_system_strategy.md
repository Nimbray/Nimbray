# Stratégie connaissances internes

Les connaissances internes doivent servir à :
- guider le raisonnement ;
- standardiser la qualité ;
- éviter les erreurs répétées ;
- renforcer les domaines pratiques ;
- documenter les décisions produit.

Elles ne remplacent pas la recherche web quand l'information est récente ou instable.
