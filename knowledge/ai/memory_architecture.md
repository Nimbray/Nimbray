# Memory Architecture — NimbrayAI V72

Une bonne mémoire IA doit être utile sans devenir intrusive. Elle doit être séparée en trois niveaux : mémoire temporaire de conversation, mémoire personnelle contrôlée par l'utilisateur, et mémoire projet.

Règles clés :
- ne jamais inventer un souvenir ;
- ne pas révéler une mémoire de façon étrange ;
- permettre l'ajout, la correction, la suppression et l'export ;
- utiliser la mémoire uniquement quand elle aide vraiment ;
- séparer les préférences personnelles des décisions projet ;
- privilégier les résumés courts et vérifiables.

La mémoire projet retient surtout : objectif, version actuelle, décisions, risques, fichiers importants et prochaines actions.
