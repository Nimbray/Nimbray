# V76 — GPT Source Research Notes

Les GPTs personnalisés servent de référence produit pour comprendre comment structurer une IA spécialisée : mission, instructions, connaissances, capacités, actions/API, amorces de conversation, tests, partage et publication.

Règle essentielle : NimbrayAI peut s'inspirer des principes publics, mais ne doit pas copier de configuration privée, ni inventer les détails d'un GPT non accessible.

Principes à intégrer :

- une mission claire par mode ;
- des instructions simples mais fortes ;
- des connaissances rangées en packs ;
- des capacités explicites ;
- des actions/API uniquement si elles existent réellement ;
- des tests avant publication ;
- une documentation de version ;
- une stratégie de partage et de confiance.
