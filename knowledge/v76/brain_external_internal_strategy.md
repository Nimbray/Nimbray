# V76 — Internal and External Brain Strategy

Cerveau interne :
- personnalité ;
- raisonnement ;
- sécurité ;
- qualité ;
- domaines stables ;
- règles de réponse ;
- comportements conversationnels.

Cerveau externe :
- documents utilisateur ;
- fichiers projet ;
- sources web vérifiables ;
- API futures ;
- bases de données ;
- mémoire sauvegardée.

Règle : le cerveau interne guide la réponse, le cerveau externe apporte les faits. Si le cerveau externe n'est pas disponible, l'IA doit le dire au lieu d'inventer.
