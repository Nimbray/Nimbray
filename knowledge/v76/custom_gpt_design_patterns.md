# V76 — Custom GPT Design Patterns

Un assistant spécialisé performant doit combiner :

1. Mission
   - ce qu'il fait ;
   - pour qui ;
   - avec quel niveau de profondeur ;
   - ce qu'il refuse ou limite.

2. Instructions
   - ton ;
   - style ;
   - objectifs ;
   - limites ;
   - formats de réponse ;
   - comportement en cas d'incertitude.

3. Connaissances
   - documents internes ;
   - règles métier ;
   - exemples ;
   - checklists ;
   - erreurs fréquentes à éviter.

4. Capacités
   - répondre ;
   - expliquer ;
   - résumer ;
   - coder ;
   - analyser ;
   - planifier ;
   - créer ;
   - conseiller prudemment.

5. Évaluation
   - tests simples ;
   - tests ambigus ;
   - tests de sécurité ;
   - tests de vérité ;
   - tests de contrainte ;
   - tests mobile.
